`.s6` 格式
---------

## 头编码

`sparse6` 格式可以表示, 0 到 2^36 - 1 阶的稀疏图.

开头可以包含: `>>sparse6<<`

接下来用 `:` 表示系数图.

## 顶点编码

接下来是节点数编码 n, 可能是 1 或 4 或 8 字节

```shell
x0
~, x1, x2, x3
~, ~, x1, x2, x3, x4, x5, x6
```

- 如果 0 <= n <= 62，则为单字节 x0 + 63。
- 如果 63 <= n <= 258047, 则为四字节, 后面三字节是个 64 进制数
- 如果 258048 <= n <= 68719476735, 则为八字节, 后面六字节是个 64 进制数
