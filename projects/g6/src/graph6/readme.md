## 合法性判定

0 到 68719476735 (2^36 - 1) 阶的简单无向图。

开头可以包含: `>>graph6<<`

一张图：
假设 G 有 n 个顶点。写邻接的上三角
G 的矩阵作为长度为 n(n-1)/2 的位向量 x，使用排序
(0,1),(0,2),(1,2),(0,3),(1,3),(2,3),...,(n-2,n-1)。

然后该图表示为 N(n) R(x)。

例子：
假设 n=5，G 有边 0-2、0-4、1-3 和 3-4。

x = 0 10 010 1001

那么 N(n) = 68 和 R(x) = R(010010 100100) = 81 99。
所以，图形是 68 81 99。
